inspiration

I developed this tool because I wanted to know who unfollowed me after I followed them. I knew of other apps that will do this for you, but I did not want to give a third-party my information for fear of being hacked. This led me to develop my own automated solution to this problem.

technology used

This project uses NodeJS and express for its backend. The data is obtained by running selinium, a web automating framework used for webscrapping, and uses bootstrap for the frontend.

development journey

  1. in-browser JavaScript for data collection and processing
  2. node+selinium for data collection and processing
  3. avoiding instagram restrictions/bot detection
  4. add frontend to replace static user auth
  5. add bootstrap to make it look good
  6. add results page for downloading the output
